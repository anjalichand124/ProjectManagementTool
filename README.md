# Project Management Tool
This is a simple Project Management Tool built using Django during my Full Stack Web Development internship. The project allows users to manage projects and tasks in an organized way.

## Features

- User Login and Logout
- Dashboard
- Create Projects
- Add, Edit and Delete Tasks
- Task Board
- Comments

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## How to Run

```bash
python manage.py runserver
```

Open the project in your browser:

http://127.0.0.1:8000/

## Project Structure

- accounts
- comments
- projects
- tasks
- project_management

This project helped me learn Django basics, authentication, CRUD operations and project organization.
